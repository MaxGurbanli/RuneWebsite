On April 4th, 2022


Version 2.0 is a massive update that has been in the works for a long time and adds the following features: Businesses, new bank levels, localized leaderboards, context menu commands, and much more.

Before we begin...
Over the past few weeks a lot of Rune's code had to be rewritten due to a library switch. This was a lot of work and the main reason behind the "v2.0" update. Unfortunately all slash commands had to be removed for now. I hope to get them back soon, so expect them to slowly return in the following updates.

Businesses
Introducing businesses! These generate revenue for you in the background, helping you continue your economy adventure!



Each business you own accumulates revenue every 15 minutes, up to 96 times before the revenue has to be collected for any more coins to be generated.



Businesses can be upgraded to increase their revenue. These stats are universal for all businesses:
- Each upgrade increases revenue/hour by 12 coins
- Each upgrade costs 800 coins



In summary, the following commands have been added:
- !!business view to view your owned businesses
- !!business list to view a list of business you can buy
- !!business buy [ID] to buy a business
- !!business upgrade [ID] to upgrade an owned business

Localized Leaderboards
Leaderboards are now fully localized, using !!leaderboard displays the richest people in the server the command is used in.
The leaderboard page has also been revamped to display more information, but now only shows the top 8 users instead of the previous 10.



More Economy changes
- Added bank levels 13, 14, and 15
- Prices for upgrading bank levels have been adjusted. View the most up to date prices using the new !!upgrades command
- The !!fish command features a more interactive task

Context Menu Commands!
- Added "User Profile" context menu command
Right click a user -> Apps -> Select "User Profile" to view information about the user!



- Added "Message Info" context menu command
Right click a message -> Apps -> Select "Message Info" to view information about the message!



Slash Commands
- The /whois command is already available to use as our first slash command since the library change!



Other changes
- Moved !!whois command to the Utility category
- Removed !!stats command

Phew, that was a lot...

That's all for now! A lot of this comes from your feedback - share your thoughts in our Discord server.
